#
# BidsApps Boutiques Descriptors
#

Place in this subdirectory all Boutiques JSON descriptors
for BidsApps, and they will be integrated in CBRAIN in a
more friendlier manner than if they were integrated using
the normal Boutiques loader.

BidsApps descriptors MUST contains a specific set of inputs
with precise IDs. The loader will complain and refuse to load
them otherwise.
